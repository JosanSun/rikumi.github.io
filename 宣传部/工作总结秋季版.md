# 16-17-2工作总结暨宣传部参考指南（秋季版）

2017/2/10丨[本文档永久链接](https://rikumi.github.io/#/宣传部/工作总结秋季版)

本指南按照时间顺序，以2016年秋季学期为例，参考上一届九系学生会宣传部窦颢工作总结，详细列举计软学生会宣传部秋季学期需要做的事，并从我个人的角度给出一系列总结和建议，供后人探讨和借鉴。**为了方便工作，一切叫得出名字的正式活动都以其（目前的）全名作为大标题。**

## 迎新宣传、招新宣讲、招新面试、素拓、风采展示等

1. 暑假期间准备的**迎新宣传**，我本人只参与了展架的设计（另外可能还有迎新文化衫需要设计），当时延续了上一届宣传部的传统，自己逼自己改了好几版，最终各方面评价都还可以。另外暑假期间需要协助新闻部完善迎新专刊，他们会定期在群里发做好的pdf让大家检查，宣传部这时候应该适当负责，及时发现其中的问题。
2. 短学期期间的**招新宣讲**准备的不算很充分，主要原因是时间不够，希望以后的副部能把招新宣讲的准备提前到暑假期间，尽量多用有创意的视频，充分展示宣传部的风采，少用传统的ppt+上台演讲的方式，本来以为这样会省事，结果背稿子和改ppt反而花了不少时间，而且传统的招新演讲有种自卖自夸的负罪感；
3. **招新面试**期间，我个人本来提倡采用一些考题，标准化地给一个应试者评分，但很快发现这样效果反而不好，所以在大家提议下做了补救，即根据考题，开放式地让应试者说自己的看法，而不是强制要求他们二选一或多选一；另外需要后人注意的是，由于短学期时间安排紧凑，招新面试不得已只能安排在中午和晚上，尤其是中午很难有时间吃午饭（从软件楼出发到各个食堂距离都比较远），所以需要特别想办法解决好伙食问题；
4. **招新培训**时，讲知识点的节奏比较快，大家基本跟不上，也听不明白我们在说什么。以后应当注意，PS等工具应当从最最基础的知识讲起，比如首先让大家充分理解图片格式、画布、图层、工具栏中的每一个工具、一些快捷键，保证大家掌握了基础知识之后，再通过实例讲解一些具体问题的处理；也不要试图在培训期间给部员灌输审美观念，因为这是需要通过长期实战才能获得的。关于培训的时间点，PS培训一般在招新后利用机房立即进行（因为这时候部员还没带电脑），Pr培训一般在校史决赛分工之前的例会上进行，AE培训则在校史决赛分工之后的例会上。
5. 关于**第一次任务（校史+自行车赛）的分工**，我们采取了先分块、再定人的办法，即先将整个任务分成以人为单位，再让大家用投票等方式选择自己想做的部分。实际操作中我们发现这样可以很快的发现哪些部员在哪些领域有特长，比如某一两个男生可能会特别喜欢制作视频，那么之后的各类活动也会重点培养他们做视频的特长；
6. **招新素拓**需要每个部门的部长和副部一起负责组织一个游戏，另外需要有一个人在后台放热场音乐，作为宣传部的副部最好自告奋勇去做这个事；
7. 招新刚结束之后一定要及时抽一个周末**集体聚餐**一次，因为过了这段黄金时间，大家都开始忙了，也很难再找时间聚餐了。另外建议聚餐时带一些比较棒的宣传部元老一起，不仅可以活跃气氛，还能传授一些非常有用的选课指南等等；
8. 关于**部门风采展示**，作为副部可以不参加，但是如果选择不参加，一定要明确选定让哪些部员负责组织展示，而不要指望让他们自告奋勇地组织活动，这是今年风采展示的一个教训。另外，如果有精力，尽量让每个人都参与进来，或者（如果是做鬼畜视频展示之类的话）至少让每个人都上镜。这可是一个开玩笑的绝佳机会，部门内部玩笑开的越棒，越容易吸引其它部门的眼球，甚至捧红一个部员。

## 东南大学2016（新生文化季系列活动之）校史校情知识竞赛

校史竞赛活动是原软院学生会继承下来的活动，因此我本人可能不是很熟悉相关的流程，经过跟其他人沟通，才逐渐了解一些细节。校史活动可以分初赛和决赛两个部分，进而可以细分成初赛宣传、初赛组织、决赛宣传、决赛组织四个部分。这四个部分几乎是完全不相关的（尤其是在时间和组织方式上）：

1. 初赛宣传是在招新之前进行的，因此只有部长和副部参与工作，加之短学期比较忙碌，所以可能没有足够的时间改稿和交流，所以最好的办法就是自己及时做好自己的事，自己强迫自己不断修改；
2. 初赛组织也是在招新之前进行，同样只有部长副部参加，属于学生会不分部门的统一任务，具体任务主要就是监考。监考期间需要向考生说明一些操作方法，另外也会遇到一些不可避免的系统问题，这时候需要利用大屏幕详细说明情况，另外遇到特殊问题也要通过手机及时汇报，所以一定要保持手机电量充足，带好充电线（尤其苹果用户，我们监考当天包括我在内至少有3个iPhone用户到处求充电线）；当天的情况和招新面试非常相似，尤其对于短学期身处软件楼的同学来说，也几乎没有吃午饭的时间，不过可以撑到一点多结束之后去梅园之类的不歇业食堂吃。
3. 校史决赛宣传是招新完成之后，全体部员和部长副部一起完成的前两项任务之一（另一个是自行车赛）。校史决赛要做的有三样东西：LED视频、开场视频、ppt。我本人当时参与了LED视频，在比较奇怪的要求下做了好几版也没让领导满意（要求是用学校的老照片，展示校史风采，而且还要大量结合学校的科研成果，这个需求是后来加的），后来好像被别人拿去改了（？），反正不了了之，剩下的时间赶紧专心投入自行车赛中了；
4. 校史决赛的组织是学生会不分部门、重新分工的任务，当天在润良报告厅，以各学院代表团之间的问答、抢答的形式为各学院排出名次并颁奖。在这期间要做的工作不多，根据各个工作组的不同，可能会有布置场地、负责工作餐、管理门厅、管理多媒体设备等等；比起组织活动，更重要的是在润良这么庄严的地方看一场校级的重要活动，收获很大。

## 东南大学第八届环九龙湖自行车大赛

自行车赛同样是分成宣传+参与两部分，其宣传部分也是极尽复杂，**首先要号召部员一起设计好一个logo，如果部员设计logo的时候感觉比较手生，可以亲自操刀；**logo出来之后，各个宣传品就有了一个统一的元素。宣传品不按个算，而是按批次，一批一批的制作和印刷：

1. 首先是**需要提前一到两周**在淘宝订购的纪念品**手环、卡贴、胸章、明信片、笔记本**这五个，其中明信片在贝兹广告也可以印，但是由于去年反馈质量不好，今年开始改成在淘宝订购。今年用到的具体商品编号、预算、数量可以在网盘上找到（/学生会/2016-2017/自行车赛/最终）。**另外我们提倡创新，这五种纪念品连续用了两年，希望下一届可以创新一下，在预算允许的范围内考虑其它的纪念品；**
2. 淘宝商品的订购方法：**先自己用淘宝联系商家，问清楚批量订购的事宜，（如果需要定制，把尺寸、色彩等要求问清楚并交给部员来做，做好后把图片跟商家发一遍）核对清楚确定可以订购之后，**再把商品链接、相关要求和可能用到的图片素材打包发给生活部，生活部会用他们的淘宝账号下单并处理好费用周转；
3. 第二批宣传品是贝兹要印刷的工作证、证书、奖牌，以及**纪念品中的书签**，这些也属于广义宣传品（不用来宣传的宣传品），需要**在比赛前一周之内**做出来，另外可能还要协助其它部门制作秩序册**（一般其它部门会把秩序册做好，不需要宣传部过于担心，但是因为也属于广义宣传品所以要负责确认一下有没有落实到位；另外注意，秩序册内页需要调整为4的倍数才不会出现空页）**，具体需要做的各类证书奖牌清单参见网盘（/学生会/2016-2017/自行车赛/最终/贝兹/）；和贝兹的沟通方法和注意事项详见附录；
4. 第三批宣传品即海报、喷绘、展架、横幅，前三样需要提前构思并设计好样式，一定要让人眼前一亮（**自行车赛的宣传品是我们受众最多的宣传品，校外来参赛的人都会看到**，因此每年自行车赛的宣传品都经过了很多次改稿，如果做得好的话会特别给计软增光）。根据传统和设计规范，海报、喷绘、展架样式统一即可，一般不需要分别设计，只需要根据不同的尺寸调整排版。**第三批宣传品需要根据体育部的策划书，确定好时间地点之后，按照体育部推荐的开始宣传时间（这个需要主动询问，一般都是比赛前一周开始宣传），并且在开始宣传之前的2-3天交付印刷**；
5. 关于横幅，只要号召大家根据要求（尺寸、挂的位置、内容大意）和需要的数量想一些长短不同的标语即可，一般要做**两个比较逗比的宣传标语、一个“预祝……圆满成功”、以及一个为计软院系加油的标语（这个一定不要忘了）**，一共四条；
6. 展架只要做宣传用的展架即可，揭示比赛规则、比赛项目等等的展架可以问体育部是否需要，如果需要，让体育部出对应的文案，再根据宣传品的整体风格来做；
7. 自行车赛举办当天，学生会全体干事重新分成多个工作组，副部和部长退居二线，指导干事进行工作，但按照要求也应当全程在场；副部应当根据自己去年参与工作的经验，主动去自己熟悉的工作组旁观，并在他们有问题的时候提供帮助。

## 计软「以梦为码，即刻编译」主题迎新晚会

1. 首先要提前一个月左右在两宣两新（学生会宣传、青协宣传、学生会新闻、科协新媒体）组织中确定好分工，尤其是各个视频（倒计时视频、可能有也可能没有的开场视频、中场视频）和抽奖软件，然后按照计划好的时间点把分配到的工作做好；
2. 设计海报和喷绘（今年迎新晚会的喷绘给我的收获就是，宣传品也不一定要网红脸，有时候走小清新等等非标准路线也可以很好看），另外根据院里的要求做一些印刷版和电子版（图片格式，发邮件用）的邀请函；
3. 大屏背景需要设计好，有一个漂亮的背景图、明亮鲜艳的配色和布局协调的logo；侧屏报幕的ppt和节目单要让部员做好，并注意审阅节目名、人名、组织名称等等有没有错误；目前焦标侧屏是9：16的竖屏，注意在做侧屏ppt的时候把页面尺寸和方向设置正确；另一个侧屏的微信墙由新闻部负责；
4. 另外，要及时跟外联部沟通，协议中要求的广告语、赞助商logo、广告视频等一定要按照要求放置和播放，并且注意如果某个logo放在宣传品上不好看，要及时进行调整；
5. 建议提前几天观看一下其他院系的迎新晚会，熟悉一下他们的模式，以便借鉴；
6. 关于主屏的播放，焦标分成道具池和音控室两个区域负责。宣传部主要需要有一名副部负责在道具池下面播放视频，所有有具体视频或者有ppt的节目都由道具池负责；其它没有视频的节目，由道具池中的贝兹广告师傅负责播放屏幕动画、音控室负责放背景音。另外，主屏背景也会提前拷贝到贝兹师傅的电脑里，当做一个静态的屏幕动画来播放，你的电脑只负责放**有视频节目的视频、有ppt节目的ppt、中场视频、选好的暖场和散场歌单、抽奖程序**，其它情况下都利用控制台切换到贝兹的电脑上播放。贝兹一般会提供一些常用的转接线例如VGA头、HDMI头等，其它插孔的电脑最好自备一条。彩排时师傅会帮你设置从两个电脑之间渐变切换，设置好之后要熟悉这个切换方法。电脑的声音输出由玩转舞台工作室负责接无线音频，如果音频有问题需要询问音控室给音了没有；无线音频如果没电，需要玩舞的同学负责换电池；
7. 一定要注意各个节目的衔接要快，而且要注意场上是否已经准备好，每一步操作前请三思。**不操作的时候，请不要（也不要让他人）随意触碰控制台、无线音频终端、在平台上剧烈跳动或晃动、甚至摇晃控制台**，否则可能导致大屏幕出现短暂黑屏、声音卡顿等问题；抽奖程序正式运行前，务必将其关闭并重新打开一次，否则可能由于各种原因无法工作；**请使用鼠标进行操作，即便你是mac用户**；如果某些节目有专人负责播放ppt，请提前让他们熟悉操作，例如你的鼠标滚轮的方向可能和他们期望的相反，也要让他们注意使用鼠标，不要误触触摸板；
8. **电脑连接屏幕时建议使用“复制”模式，而不是“扩展”，否则在这种特殊情况下会很难操控。**对于某些有ppt的节目，**一定要检查他们的ppt的“幻灯片放映选项”中是否勾选了“使用演示者视图”，如果勾选了请务必去掉**，因为这样会在播放时强制让视频输出从“复制”模式变成“扩展”模式，而且播放结束后不会自动还原，从而为后续播放带来困扰；
9. 另外要随时记住在道具池中放了衣服、包裹、电脑、手机、U盘、台本的人，因为这些东西很容易遗忘而且很难找到失主。
